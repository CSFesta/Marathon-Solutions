- 🧩 **Dificuldade:** 4 / 12 
    -
- 🗂 **Tópicos:** [**dfs, grid**]  
    -
- 🧮 **Complexidade:** `O(N * M)`
    -
- 🎯 **Desafios principais:** 
    - Sabendo fazer um dfs recursivo numa grid, o problema se torna facil
- 🔎 **Ideia:**
    - Ilha = quantos valores iguais são possiveis de alcancar apenas, indo para esquerda, direita, em cima ou em baixo.
    - Para cada ilha ainda não visitada:
        - Verificar o tamanho da ilha atual 
        - Descobrir qual o maior tamanho de uma ilha
